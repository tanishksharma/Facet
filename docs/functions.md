# Functions

The API index. One line per call and per attribute. Each entry links to its component on Elements or Layouts; the component is described there, never here.

The rule: a function consumed by only one component belongs to that component. It is plain JavaScript, wires itself from the markup, and sits under Component calls. Page-level calls are the true API.

## Page-level calls

- `facet.set({...})`. Change theme, mode, language, width, any token, live. [done]
- `facet.scheme.set("auto")`. The light, dark, auto control. [done]
- `facet.translate()`. Re-apply translation to late markup. [done]
- `facet.locate()`. The opt-in coarse locale helper. [done]
- `facet.groupNumber(n)`, `facet.numberWords(n)`, `facet.numberSystem()`. Locale number helpers. [done]
- `facet.go(url)`. Navigate with the page transition. [done]
- `facet.toast(message, kind)`. Fire a toast. [done]
- `facet.busy(el, true)`. The loading skin. [done]
- `facet.icon()`. Fill icon glyphs. [done]
- `facet.motion.register(sel, opts)`. Add elements to the motion engine. [done]
- `facet.feedback.tap()`, `.tick()`, `.snap()`, `.success()`. Sounds and haptics. [done]
- `facet.install()`. The install prompt. [done]
