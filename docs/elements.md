# Elements

Every component, one line each. States, themes and sizes are system facts on the Features page and apply to everything here. Every entry links to the elements that look like it and to the function that drives it.

## Layout primitives

- Container. A page-width wrapper that centers content and caps its width.
- Stack. Vertical flow with a fixed gap.
- Row. Things side by side that wrap.
- Grid. Auto-fitting columns, no breakpoints.
- Snap pager. Full screens, one at a time. (Linked: view stack, Layouts shells.)
- View stack. App screens, one visible, Back works. (Linked: snap pager.)

## Form controls

- Button. A clickable action.
- Checkbox. A yes or no tick.
- Radio. Pick one, all options visible.
- Switch. A setting that applies the moment it flips.
- Stepper. A number between minus and plus buttons.
- Segmented control. A pill of options, exactly one pressed.
- Field. A labelled wrapper around any input, hint line beneath.
- Field actions. In-field clear plus a menu: copy, paste, select all, undo, redo.
- Password reveal. A show-hide eye.
- Password strength. A live strength bar under a password field.
- File upload. The native picker, themed, with a drop zone.
- Date input. The native picker, themed.
- Time input. The native picker, themed.
- Date range. A start and end date as one control.
- Memorable date. Day, month, year as three fields.
- Number input. Groups digits per locale, reads the value in words.
- Slider. Drag a number in a range, live readout.
- Detailed slider. Slider plus steppers plus a number box.
- Dual-thumb range. One track, two handles, picks a range.
- Choice grid. Square buttons, one exclusive pick.
- Combobox. An input that filters a list as you type.
- Multi-select. Several picks shown as removable chips in the field.
- Tags input. Type and Enter makes a chip.
- OTP input. One-digit boxes for verification codes.
- Rating. Stars as input, and the read-only twin.
- Character count. Live count under a limited field.
- Input add-ons. Fixed prefix or suffix: currency, units, domains.
- Conditional reveal. A choice reveals its follow-up field.
- Inline edit. Click a value, it becomes a field.
- Copy button. Copies an element's text.

## Content pieces

- Card. A padded rounded panel grouping one thing. (Linked: list, feature grid, stat row.)
- Feed post. A card for one authored post.
- Entry. One dated item: a job, a release, a milestone.
- Testimonial. A quote with its author.
- List. Repeated rows with icon, text and trailing value. (Linked: card, description list, table.)
- Description list. Labelled key-value rows. (Linked: invoice, settings section.)
- Task list. A checklist with per-row status.
- Comment thread. Nested comments with reply actions.
- Chat bubble. Sent and received messages with timestamps.

## Overlays

- Modal. A dialog for confirmations and short tasks.
- Drawer. A panel from a screen edge.
- Popover. A small anchored bubble, closes when attention moves.
- Popconfirm. A tiny are-you-sure popover on the button itself.
- Accordion. A fold for optional reading.
- Toast. A transient notice that leaves on its own.
- Dropdown menu. A few actions behind one button.
- Context menu. Right-click actions on app surfaces.
- Nested submenu. A menu item that opens a second level.
- Command palette. Fuzzy search over actions and pages, on one shortcut.
- Hover card. A rich preview panel on hover or focus.
- Banner bar. A page-top strip for announcements, dismissible.

## Navigation

- Tabs. Buttons that switch views in place.
- Steps indicator. Numbered stages of a multi-step flow.
- Breadcrumb. The path to the current page.
- Back link. A small back step above the title.
- Back to top. Appears after scrolling, returns to the top.
- Scroll return. After any jump to top, a button returns you exactly where you were.
- Pagination. Numbered links through a long set.
- Nav link. The link style that marks the current page.
- Subscribe. A compact email sign-up.
- Mega menu and footer live on the Layouts page as blocks.

## Data display

- Table. Real records, sortable columns, restacks as cards on phones.
- Bulk actions. Select rows, a bar shows the actions.
- Tree view. An expandable hierarchy.
- Badge. A word in a pill stating a status.
- Chip. A small toggleable pill, with a dismissible variant.
- Avatar. An image or initials, round.
- Avatar group. Overlapping avatars with an overflow count.
- Progress. A completeness bar.
- Progress ring. The circular twin with a center value.
- Icon badge. A status dot on an icon corner.
- Kbd. A keycap for shortcuts.
- Countdown. Live time remaining to a moment.
- Watermark. A faint repeated label over content.
- The comparison table lives on the Layouts page as a block.

## Charts

One family, one voice: which chart for which data is on the Guidance page.

- Line. One series over time.
- Bar. Categorical comparison.
- Area. The line with a filled wash.
- Donut. Part of a whole, center stat.
- Sparkline. A tiny inline trend.
- Meter. A capacity bar with zones.
- Gauge. The dial version.
- Heatmap. Day cells colored by intensity.

## Status

- Skeleton. A placeholder shaped like coming content.
- Spinner. A ring for unknown waits.
- Empty state. Names what would be here, offers the filling action.
- Loading skin. One attribute turns a region's own content into placeholders.
- Result block. A verdict line, one big number, the supporting figures.

## Media

- Video and audio. The native players, themed like everything else.
- Lightbox. An image opens full screen with next and previous.
- Pinch-out image zoom. Pinch any image and it grows out of its container, one layer up, and settles back; the page itself never zooms.
- Gallery grid. Thumbnails that open the lightbox.
- Carousel. Slides with arrows, dots and optional autoplay.
- Aspect ratio. Locks a box to 16:9, 1:1, 4:3.
- Before-after slider. A draggable divider between two images.
- Device frames. Phone and browser mockups for screenshots.

## Text utilities

- Flagship link. A large link for the one onward step.
- Heading bar. A short accent stroke under a heading.
- Section title. The section-opening heading style.
- Eyebrow. The small spaced-caps label above a heading.
- Callout. A bordered aside that breaks information down.
- Divider with label. A rule carrying centered text.
- Text clamp. Holds text to a line count with a soft fade.
- Description tooltip. A one-attribute explanation bubble.
- Visually hidden. Screen-reader-only text.

## Decoration

- Backgrounds. Grid, fluid pools, and the aero scene.
- Adaptive ink on glass. Text on glass flips light or dark with what is behind it.
- Reflection. A soft mirror below an element.
- Go arrow. A trailing arrow that slides forward.
- Shine. A specular light that travels with the device.
- Scroll-edge fades. Gradients that signal more content.
- Themed map. A map colored by the active theme.
- Annotation kit. Margin notes, hand-drawn circles, highlighter, stamps.
- Icons. 108 and growing line glyphs, filled from one attribute.

## Effects

One family of small, opt-in animation pieces, each honoring reduced motion: spotlight card, magnet, click spark, dock magnify, dot field, scroll reveals, split text, blur text, count up, typewriter text, rotating text, scramble text, shine button, traveling border, glare hover, gradient text, shiny text, glitch text, mac-window code block, splash screen.
