Facet

A design library in plain HTML, CSS and JS

Facet styles everything I make: apps, websites, pitch decks, documents and business cards. It is one CSS file and one JS file at a public URL. No React, no build step, no npm install, never minified. This site is built with it, from it, and doubles as its documentation.

Everything else

Spacing runs on a 4px base (--space-1 to --space-8), container widths come as narrow, default and wide, radii as small, medium, large and pill, shadows as flat, raised, floating and overlay — whisper-flat, all of them — and motion as fast, normal and slow with two easing curves. Read facet.css: it is the readable, commented source, and the full token list lives at the top of it.
